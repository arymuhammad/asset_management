# assets_management

Asset Management adalah web app untuk me manage setiap asset perusahaan baik yang di HeadOffice dan juga di seluruh store URBAN&CO. meliputi menu Dashboard, Master Kategori, Master Asset, Stok, Barang Masuk, Barang Keluar, Adjusment In, Adjusment Out, Stock Opname, Report Maintenance, Request Form.
